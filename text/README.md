Source text for media assets — research briefs and narration scripts.

## Naming convention

```
YYYY-MM-DD_topic[_suffix].md
```

- **`_narration`** — the narration script (front-matter + `!SLIDE:` directives) used to build a video.
- **no suffix / descriptive** — the underlying research brief or source document.
