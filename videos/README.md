# videos/

Rendered video files (`.mp4`).

## Naming convention

```
YYYY-MM-DD_topic-in-kebab-case.mp4
```

- **`YYYY-MM-DD`** — render date (ISO 8601, sorts chronologically).
- **`topic-in-kebab-case`** — short descriptive slug, lowercase, hyphen-separated.

Example: `2026-07-19_all-of-us-professor-briefing.mp4`
