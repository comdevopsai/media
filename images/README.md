# images/

Images, poster frames, thumbnails, and charts.

## Naming convention

```
YYYY-MM-DD_topic-in-kebab-case[_suffix].png
```

- **`YYYY-MM-DD`** — creation date.
- **`topic-in-kebab-case`** — short descriptive slug.
- **`_suffix`** (optional) — role, e.g. `_thumb`, `_chart`, `_poster`.

Example: `2026-07-19_all-of-us-professor-briefing_thumb.png`
