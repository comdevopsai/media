# presentations/

Slide decks and presentation sources (narration scripts, chart specs, generated slides).

## Naming convention

```
YYYY-MM-DD_topic-in-kebab-case/          # one folder per presentation
├── narration.md                          # narration script (!SLIDE: blocks)
├── charts_spec.txt                        # make_charts.py stat spec (if any)
└── slides/                                # generated slide/chart PNGs
```

Or, for a single-file deck: `YYYY-MM-DD_topic-in-kebab-case.pdf` / `.pptx`.

- **`YYYY-MM-DD`** — creation date.
- **`topic-in-kebab-case`** — short descriptive slug.
